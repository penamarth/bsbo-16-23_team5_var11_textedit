### Сценарий использования: Успешное сохранение файла

Успешное сохранение (PDF) — текстовый сценарий
1.	Пользователь в интерфейсе выбирает «Сохранить как», указывает формат pdf и путь /path/to/file.pdf.
2.	Editor принимает команду exportCurrent(format="pdf", path="/path/to/file.pdf").
3.	Editor получает текущий Document и его baseDoc (DocxDocument).
4.	Editor создаёт PdfAdapter, передавая ему DocxDocument (реализует IExportable).
5.	Editor вызывает у Document метод exportAs(adapter=PdfAdapter, path="/path/to/file.pdf").
6.	Document делегирует export(path) объекту PdfAdapter.
7.	PdfAdapter запрашивает DocxDocument.getContent() → получает строку, например: "The data is stored in the database...".
8.	PdfAdapter конвертирует содержимое в PDF и сохраняет файл по пути /path/to/file.pdf; возвращает успех в Document.
9.	Document подтверждает успех Editor, а Editor показывает пользователю уведомление: «Файл сохранён: /path/to/file.pdf».
