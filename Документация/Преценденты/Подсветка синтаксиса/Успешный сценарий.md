1. Пользователь открывает документ
2. Выбирает в интерфейсе тип хайлайтера(что он будет подсвечивать), тем самым вызывая нужный метод у класса Document
3. Сам хайлайтер метод проверяет весь документ и применяет форматирование на месте
4. Хайлайтер метод вызывает метод draw() у ViewText
5. У пользователя отображается подсвеченный текст
