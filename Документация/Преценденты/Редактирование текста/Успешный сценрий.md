# Успешный сценарий

1. Пользователь через пользовательский интерфейс вводит текст, который необходимо вставить
2. У класса-контроллера Editro вывзвается метод InsertText("Текст пользователя")
3. Далее работает последовательный алгоритм:
                       1. У Document вызывается метод InsertText("Текст пользователя")
                       2. Document.InsertText("...") разбивает текст на абзацы. Создаёт объекты ParagraphComponent, вызывает у них метод add()
                       3. ParagraphComponent.add("...") разбивает абзац на предложения. Создаёт объекты SentenceComponent, вызывает у них метод add()
                       4. SentenceComponent.add("...") разбивает предложение на слова. Создаёт объекты WordComponent
                       5. SentenceComponent.add("...") завершает работу, добавляя список обхектов WordComponent
                       6. ParagraphComponent.add("...") завершает работу, добавляя список обхектов SentenceComponent
                       7. Document.InsertText("...") завершает работу, добавляя список обхектов ParagraphComponent
4. Алгоритм успешно отработал, текст добавлен

Стуктура Document.content после вставки текста :
  Document

    ├── ParagraphComponent 1
    │   └── List<SentenceComponent>
    │       ├── SentenceComponent 1

    │       │       ├── WordComponent ("Привет,")
    │       │       └── WordComponent ("мир!")
    │       └── SentenceComponent 2 
    │           └── List<WordComponent>

    │               ├── WordComponent ("вас")

                    ├── WordComponent ("обязательно")
